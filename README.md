Este análisis tiene como objetivo identificar la mejor tienda para que el Sr. Juan venda sus productos, basándose en diversas métricas clave de rendimiento. Se evaluaron cuatro tiendas diferentes para determinar cuál ofrece el mayor potencial comercial y rentabilidad.

## Estructura del Proyecto

El proyecto se organiza en un cuaderno Colab que realiza el siguiente flujo de trabajo:

1.  **Importación de Datos**: Carga de cuatro conjuntos de datos (tienda_1.csv, tienda_2.csv, tienda_3.csv, tienda_4.csv) utilizando la librería `pandas`.
2.  **Conocimiento de la Estructura de Datos**: Exploración inicial de las dimensiones de los DataFrames (`shape`) y los tipos de datos (`dtypes`) para asegurar la integridad de los datos.
3.  **Análisis de Facturación**: Cálculo y visualización de los ingresos totales generados por cada tienda.
4.  **Ventas por Categoría**: Análisis de la cantidad de ventas por categoría de producto en cada tienda.
5.  **Calificación Promedio de la Tienda**: Cálculo de la calificación promedio de los clientes para cada tienda.
6.  **Productos Más y Menos Vendidos**: Identificación de los productos con mayor y menor volumen de ventas en cada tienda.
7.  **Costo de Envío Promedio por Tienda**: Determinación del costo de envío promedio para cada establecimiento.
8.  **Conclusión y Recomendación**: Integración de todos los hallazgos para formular una recomendación al Sr. Juan.
